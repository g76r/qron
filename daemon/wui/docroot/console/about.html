Qron: the free scheduler

Qron is an enterprise-grade scheduler with time-based and event-based triggers, dependencies between tasks, resources management, monitoring, rest api, web console, and even more features. And it's free software.

About Tools & Libraries

Qron uses, embeds or includes several other free software. The most proheminent ones being: Qt, Twitter Bootstrap, jQuery, AnchorJS, OpenSSH and Graphviz.

Qron uses, embeds or includes several free icons from these collections: Font Awesome, MFG Labs, Entypo, Linecons, Modern Pictograms, Maki, Elusive and Web Symbols. To be used in the web console, the icons are compiled into a single icons font thanks to Fontello.
